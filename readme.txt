== Description ==

Inspired by Yelp compliments system BuddyPress Compliments is a BuddyPress addon that allows members to send each other compliments or eGifts. This plugin allows you to connect myCred with BuddyPress Compliments and award users points for giving or receiving compliments.

= Built-in Support =
Here are the most notable features of myCred for BuddyPress Compliments plugin.

* Award points for giving compliments.
* Award points for receiving compliments.
* Badge support.
* Translation ready.

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for BuddyPress Compliments
3. Find myCred for BuddyPress Compliments and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
